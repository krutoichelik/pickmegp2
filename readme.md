Проект: Анализ эффективности инвестиций в футбольные клубы Европы

Описание проекта

Данный проект направлен на исследование взаимосвязей между финансовыми вложениями в состав футбольных клубов и их спортивными результатами, а также популярностью среди болельщиков. Целевой результат — аналитическая база данных и визуализации, которые позволяют выявить, что влияет на результативность и популярность футбольных клубов.

Бизнес-задача

Проблема: инвесторы и менеджеры клубов хотят понимать, насколько эффективно распределяются ресурсы — действительно ли большие расходы на состав гарантируют спортивный успех и рост интереса фанатов.

Цель: создать аналитическую модель, которая покажет, какие факторы (финансовые, спортивные, демографические, маркетинговые) сильнее всего влияют на среднее количество очков за матч (avg_pts), отражающее эффективность клуба.

Задачи проекта:
  1.  Собрать данные о матчах, составах и посещаемости из открытых источников.
  2.  Объединить спортивные, финансовые и маркетинговые показатели.
  3.  Оценить влияние инвестиций и возраста клуба на результаты.
  4.  Сформулировать выводы и рекомендации для инвесторов и аналитиков.

Источники данных

Globalsportsarchive.com — результаты матчей (HTML, динамические данные).
Transfermarkt.world — посещаемость матчей (HTML, статические таблицы).
Field-hub.online API — финансовые характеристики клубов (JSON API).
Wikipedia API — год основания клуба (JSON API).

Данные объединены в единый аналитический датасет.

Обоснование бизнес-задачи

Современный футбольный рынок рассматривается как инвестиционная экосистема, где спортивный результат тесно связан с экономическими показателями. Понимание зависимости между стоимостью состава, результатами на поле и популярностью клуба позволяет:

оценить окупаемость вложений;
выявить «эффективные» клубы с высоким результатом при умеренных затратах;
определить наиболее перспективные лиги и сезоны для инвестиций.

Целевая переменная и типы признаков

Целевая переменная: avg_pts — среднее количество очков за матч (учитывает победы и ничьи).

Типы признаков:

Спортивные: wins, draws, losses, goals_total, games (целые числа).
Финансовые: total_cost, team_cost_dif, avg_age (вещественные числа).
Состав команды: players_amount, foreign_players_amount (целые числа).
Популярность: avg_attendance, total_attendance, sellouts (вещественные числа).
Исторические: year_of_foundation (целое число).
Категориальные: league, team, season (строки).

Гипотезы

Этапы проекта
  1.  Выбор источников данных и формулировка бизнес-задачи.
  2.  Извлечение данных посредством API.
  3.  Извлечение данных посредством веб-скрапинга и парсинга.
  4.  Первичная обработка данных (EDA).
  5.  Получение и агрегация результатов, финальные выводы.
